# вқӨдёәд»Җд№Ҳ 0.1 + 0.2 != 0.3

еҺҹеӣ пјҢеӣ дёә JS йҮҮвҪӨ IEEE 754 еҸҢзІҫеәҰзүҲжң¬пјҲ 64 дҪҚпјүпјҢе№¶дё”еҸӘиҰҒйҮҮвҪӨ IEEE 754 зҡ„иҜӯвҫ”йғҪжңүиҜҘй—®йўҳ

TIP

жҲ‘д»¬йғҪзҹҘйҒ“и®Ўз®—жңәжҳҜйҖҡиҝҮвјҶиҝӣеҲ¶жқҘеӯҳеӮЁдёңв»„зҡ„пјҢйӮЈд№Ҳ 0.1 еңЁвјҶиҝӣеҲ¶дёӯдјҡиЎЁзӨәдёә

// (0011) иЎЁзӨәеҫӘзҺҜ 0.1 = 2^-4 * 1.10011(0011)

жҲ‘д»¬еҸҜд»ҘеҸ‘зҺ°пјҢ 0.1 еңЁвјҶиҝӣеҲ¶дёӯжҳҜ вҪҶйҷҗеҫӘзҺҜзҡ„вјҖдәӣж•°еӯ—пјҢе…¶е®һдёҚеҸӘжҳҜ0.1

е…¶е®һеҫҲеӨҡвј—иҝӣеҲ¶вј©ж•°вҪӨвјҶиҝӣеҲ¶иЎЁзӨәйғҪжҳҜ вҪҶйҷҗеҫӘзҺҜзҡ„гҖӮиҝҷж ·е…¶е®һжІЎд»Җд№Ҳй—®йўҳпјҢдҪҶжҳҜ JS йҮҮвҪӨзҡ„жө®зӮ№ж•°ж ҮеҮҶеҚҙдјҡиЈҒеүӘжҺүжҲ‘д»¬зҡ„ж•°еӯ—гҖӮ

IEEE 754 еҸҢзІҫеәҰзүҲжң¬пјҲ64дҪҚпјүе°Ҷ 64 дҪҚеҲҶдёәдәҶдёүж®ө

  • з¬¬вјҖдҪҚвҪӨжқҘиЎЁзӨәз¬ҰеҸ·
  • жҺҘдёӢеҺ»зҡ„ 11 дҪҚвҪӨжқҘиЎЁзӨәжҢҮж•°
  • е…¶д»–зҡ„дҪҚж•°вҪӨжқҘиЎЁзӨәжңүж•ҲдҪҚпјҢд№ҹе°ұжҳҜвҪӨвјҶиҝӣеҲ¶иЎЁзӨә 0.1 дёӯзҡ„ 10011(0011)
